DESCRIPTION :
A full-stack travel booking web application that allows users to explore destinations, plan trips, and manage bookings. Built using React.js for the frontend, Java and Node.js for the backend, and MySQL for the database.

FEATURES:

-Browse popular travel destinations

-Plan trips with start and return dates

-Automatic trip duration calculation

-Book trips and manage bookings

TECHNOLOGIES USED:

Frontend: React.js, HTML, CSS, Bootstrap

Backend: Java, Node.js, REST APIs

Database: MySQL

Routing: React Router

INSTALLATION & SETUP:
1. Clone the repository:
git clone <your-repo-url>
2. Navigate to the project folder:
cd project
3. Install dependencies:
npm install
4. Start the app:
npm start

Usage

1. Open the app in your browser


2. Browse destinations and explore details


3. Go to “Plan a Trip” and select places to visit


4. Choose start date and see calculated return date


5. Book your trip and manage your bookings
